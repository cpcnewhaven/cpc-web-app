## Usage Notes

- All GCS URLs (`storage.googleapis.com/cpc-public-website/...`) are publicly accessible.
- URL-encode spaces as `%20` when used inside CSS `url()` or HTML `src` attributes.
- The `[` and `]` in some filenames are encoded as `%5B` / `%5D` respectively.
